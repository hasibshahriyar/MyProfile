## ğŸ¨ Customization

1. Colors and themes can be modified in CSS variables:
   ```css
   :root {
       --primary-color: #ff5722;
       --secondary-color: #ff7043;
       --accent-color: #ff9800;
       /* ... other variables ... */
   }
   ```

2. Particle animation can be adjusted in `scripts.js`:
   ```javascript
   const particleSettings = {
       color: '#ff5722',
       count: 50,
       speed: 0.5
       // ... other settings ...
   };
   ```
